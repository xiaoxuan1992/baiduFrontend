SWS

SWS即show whachu show，年轻人自由表达个性的平台。 这里有最棒的限量版创意产品，永不停歇地推出； 这里是中国新锐人群的摇篮，投稿、学习、交流、成名； 这里是创意发烧友的圈子。
SWS意图 SWS致力于创造中国创意产品领域第一品牌，成为一个受人尊敬的企业，一个广受爱戴的创意品牌：给中国无数富有天赋的年轻人以展示才华的机会；给偏好各异的个性消费者以宽泛自由的选择；给所有会员以卓越的产品及完美的体验。 SWS的团队凝聚了创意、IT、文化、影视、广告、企划、艺术培训、夜店管理、Party策划等各方面的专业人才。由于创意市场良莠不齐的混乱格局，SWS整合各方人才精英，从创意的征集、开发到市场服务保障，都以最高标准来定义行业规范。 SWS娱乐创新
SWS创意社首先是一场365天不间断的创意比赛。这里每天都有人在企划、运筹，每天都有人在为别人的设计精心制作，sws每周评选一次优秀创意，获奖者将获得sws的创意大礼包外加若干金币及sws专用获奖证书。
“伙伴”提供给你与sws共同发展，共享成果的机会。你可以成为sws的创意大使，传播创意种子，收获创意赏金。你也可以加入sws网站联盟或商家联盟，享受创意产业的巨大商机。 艺术——SWS所有产品均来自会员设计师的创意设计，每件作品都是一件原创的艺术品。
限量——SWS所有产品均限量生产，不仅具有稀缺性，更具有收藏价值。
分享——SWS为年轻人提供一个分享和交流的社交平台，他们有相同的爱好、情趣与观念，一个平等、自由的共享舞台，可以收获来自产品以外的附加价值。与SWS有关的所有消费者与参与者，都将从中获得超值受益，一件简单的创意消费，却有来自物质与精神的多重价值：收藏、参与、分享、学习、认同、愉悦等。
